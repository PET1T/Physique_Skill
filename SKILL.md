### RÈGLE 12 — FILTRE PASSE-BAS : MÉTHODE POUR TROUVER N₁ (fréquence de coupure)
```
La fréquence de coupure N₁ d'un filtre passe-bas actif se détermine
TOUJOURS via le déphasage entre u_S et u_E, PAS via la période T lue directement.

Méthode obligatoire :
  Δφ = −(2π/T) × Δt   [Δt = décalage horizontal entre les deux courbes]
  Si Δφ = −3π/4  →  fréquence de coupure  →  N₁ = 1/T

❌ INTERDIT : lire T sur la courbe et faire N₁ = 1/T directement sans passer par Δφ.
```

### RÈGLE 13 — TRANSMITTANCE T₀ : ORDRE NUMÉRATEUR/DÉNOMINATEUR
```
T = U_Smax / U_Emax   (sortie au numérateur, entrée au dénominateur)

À la fréquence de coupure : T₁ = U_Smax/U_Emax  (valeur < 1 si atténuation)
Pour trouver T₀ :  T₁ = T₀/√2  ⟹  T₀ = T₁ × √2

⚠️ EXEMPLE : Si U_Emax = 7,1 V et U_Smax = 10 V
   → T₁ = 10/7,1 = 1,41   (PAS 7,1/10)

Vérifier toujours quel signal est l'entrée et lequel est la sortie avant de calculer T.
```

### RÈGLE 14 — IDENTIFICATION DE LA COURBE u_S DANS UN FILTRE PASSE-BAS ACTIF
```
La courbe u_S(t) s'identifie par son DÉPHASAGE (avance de phase), pas par son amplitude.

✅ Justification correcte :
   "Pour un filtre passe-bas actif, u_S est en avance de phase sur u_E
    ⟺ courbe (1) = u_S"

❌ INTERDIT :
   "La courbe (1) a une amplitude plus petite donc c'est u_S"
   → justification incomplète/fausse
```

### RÈGLE 16 — CALCUL DE LA CONSTANTE K (réaction acide-base entre deux acides)
```
K = Ka₁ / Ka₂

Si K > 1  ⟺  l'acide 1 (donneur de protons) est plus fort que l'acide 2.

Formulation en 1 ligne :
  "K = Ka(HF)/Ka(HCOOH) > 1  ⟺  HF est plus fort que HCOOH."

❌ INTERDIT : développer K = 10^(pKa₂ − pKa₁) avec dérivation intermédiaire.
```

### RÈGLE 17 — CALCUL DE m DANS UN OSCILLATEUR MÉCANIQUE (méthode préférée)
```
Méthode via énergie cinétique maximale (méthode officielle) :
  ½mVmax² = E_total  ⟹  m = 2 × E_total / Vmax²

Méthode via ω₀² = k/m est valide mais moins directe graphiquement.

⚠️ Utiliser π² = 10 SEULEMENT si ω₀ n'est pas une valeur exacte comme 4π.
   Si ω₀ = 4π : ω₀² = 16π² et π² ≈ 10 est acceptable.
```

### RÈGLE B — ÉQUATION DIFFÉRENTIELLE CIRCUIT RL : FORME NORMALISÉE OBLIGATOIRE
```
La forme OFFICIELLE de l'EDO pour u_R(t) est TOUJOURS :

  du_R(t)   R+r         R
  ------- + ─── u_R  =  ─── E
    dt        L           L

Étapes obligatoires (dans cet ordre) :
  1. Schéma fléché avec sens de i (MENTIONNER explicitement)
  2. Loi des mailles : u_R(t) + r·i(t) + L·di/dt − E = 0
  3. Substitution : u_R = R·i  →  i = u_R/R  →  di/dt = (1/R)·du_R/dt
  4. Mise en équation → forme normalisée ci-dessus

❌ INTERDIT : écrire E = u_R + u_L comme point de départ
❌ INTERDIT : laisser L/R au numérateur (forme non normalisée)
```

### RÈGLE A2020 — LECTURE DU COEFFICIENT STŒCHIOMÉTRIQUE SPATIAL (d_AB = k×λ)
```
AVANT d'écrire λ = d_AB, compter VISUELLEMENT le nombre de demi-longueurs d'onde
entre les deux points sur la figure, puis écrire :
  "d_AB = k × λ  d'où  λ = d_AB / k"
  avec k = 1 (crêtes consécutives), k = 1,5 (crête → creux → crête), k = 2, etc.

✅ OBLIGATOIRE :
  Identifier k sur la figure → écrire λ = d_AB / k avec k explicité

❌ INTERDIT :
  Écrire directement λ = d_AB sans justifier k

Exemple (Session Contrôle 2020, Physique Ex2 Q2b) :
  d_AB = 12 mm ; A et B séparés par 1,5 longueurs d'onde
  ⟹ λ = 12 / 1,5 = 8 mm   (et NON λ = 12 mm)
```

### RÈGLE B2020 — MÉTHODE POUR TROUVER φ₀ PAR LA RELATION y_P = y_O(t − θ₁)
```
Étape 1 : Lire θ₁ = d/v sur la figure (premier instant où P commence à vibrer)
Étape 2 : Écrire y_P(t) = y_O(t − θ₁) pour t ≥ θ₁
Étape 3 : Substituer et identifier φ₀ par identification des termes
Étape 4 : Vérifier par y_P(t₁) = valeur lue sur le graphe

✅ OBLIGATOIRE :
  Passer obligatoirement par la relation y_P(t) = y_O(t − θ₁)

❌ INTERDIT :
  Trouver φ₀ en posant y_O(0) = 0 ou y_O(0) croissant sans passer par cette relation

Exemple (Session Contrôle 2020, Physique Ex2 Q4c) :
  θ₁ = d/v = 10×10⁻³ / 0,4 = 2,5×10⁻² s
  y_P(t) = y_O(t − θ₁)  pour t ≥ 2,5×10⁻² s
  3.10⁻³ sin(100πt + φ_P) = 3.10⁻³ sin(100π(t − θ₁) + φ₀)
  ⟹ φ_P = φ₀ − π/2 = π/2  ⟹  φ₀ = π rad
```

### RÈGLE C2020 — CALCUL DE φ_P PAR CONDITION INITIALE y_P(θ₁)
```
Lire y_P(θ₁) sur la figure. Écrire :
  y_P(θ₁) = a sin(100π θ₁ + φ_P) = valeur_lue
Résoudre pour φ_P, puis vérifier le signe avec la pente du signal au même instant.

✅ OBLIGATOIRE :
  Calculer φ_P par condition initiale + vérifier avec la pente

❌ INTERDIT :
  Poser φ_P = 0 ou φ_P = π/2 sans calcul ni justification

Exemple (Session Contrôle 2020, Physique Ex2 Q4b) :
  y_P(t = 3×10⁻² s) = −3×10⁻³ m  (lu sur figure-7)
  sin(100π × 3×10⁻² + φ_P) = −1
  sin(π + φ_P) = −1  ⟹  π + φ_P = −π/2  ⟹  φ_P = π/2 rad
  ⟹ y_P(t) = 3.10⁻³ sin(100πt + π/2)  pour t ≥ 2,5×10⁻² s
```

### RÈGLE E2020 — LECTURE DU GRAPHE RC : MÉTHODE DE LA PENTE AVEC DEUX POINTS EXPLICITES
```
Pour lire la pente d'une droite Δt = f(R) :
  1. Choisir DEUX points bien distincts sur la droite (pas forcément des points mesurés)
  2. Écrire : "D'après la figure-X, pour R = ... kΩ : Δt = ... ms
               et pour R = ... kΩ : Δt = ... ms"
  3. Calculer : pente = ΔΔt / ΔR avec les unités explicites

✅ OBLIGATOIRE :
  Deux points explicites + calcul de pente avec unités

❌ INTERDIT :
  Lire un seul point et diviser par R
  (erreur si la droite ne passe pas par l'origine exacte)

Exemple (Session Contrôle 2020, Physique Ex1 Q2a) :
  Pente correcte : a = 25×10⁻⁶ s/Ω = 25 μs/Ω
  ⟹ Δt = 25×10⁻⁶ × R   (R en Ω, Δt en s)
  ⟹ C = Δt / (5R) = 25×10⁻⁶ / 5 = 5×10⁻⁶ F = 5 μF
```

### RÈGLE G2020 — EFFET DE LA TEMPÉRATURE SUR L'ÉQUILIBRE : CITER LA LOI DE MODÉRATION
```
Structure obligatoire en 3 étapes :
  Étape 1 : τ_{f2} < τ_{f1}  ⟹  x_{f2} < x_{f1}  ⟹  réaction inverse favorisée
  Étape 2 : "D'après la loi de modération, le système évolue dans le sens

  Étape 3 : Conclusion : θ₂ > θ₁

✅ OBLIGATOIRE :

## MODULE P4 — OSCILLATIONS MÉCANIQUES (PENDULE ÉLASTIQUE)

### P4.1 — Qualifier les oscillations
```
"Libres amorties" : amplitude décroissante, sans excitateur extérieur
"Libres non amorties" : amplitude constante, sans frottements
"Forcées" : excitateur présent, fréquence imposée

### P4.2 — Équation différentielle (2ème loi de Newton)
```
Forces : F_ressort = −kx  (seule force selon Ox)
2ème loi : m × d²x/dt² = −kx
→ d²x/dt² + (k/m) × x = 0
→ d²x/dt² + ω₀² × x = 0    avec ω₀ = √(k/m)   ✓

Solution : x(t) = X_m × sin(ω₀t + φ_x)
```

### P4.3 — Lire X_m, T₀, φ_x sur la courbe
```
Sur la courbe x = f(t) :
- X_m = valeur maximale sur axe Y
  "D'après la figure 4, X_m = 8 cm"

- T₀ = distance entre deux maxima consécutifs sur axe X
  "D'après la figure 4, Δt = 1,6 s correspond à 2T → T₀ = 0,8 s"

- φ_x : à t = 0, lire x(0) = X_m × sin(φ_x)
  Si x(0) = 0 et courbe monte → φ_x = 0
  Si x(0) = X_m → φ_x = π/2
  Si x(0) = 0 et courbe descend → φ_x = π
```

### P4.4 — Calcul de k (raideur)
```
ω₀ = 2π/T₀  et  ω₀ = √(k/m)
k = 4π² × m / T₀²

A.N avec π² = 10 :
k = 4 × 10 × m / T₀²
```

### P4.5 — Énergie mécanique
```
E = ½k × x²(t) + ½m × v²(t)

Sans frottements : E = constante = ½k × X_m²
Calcul de v₀ : v₀ = √[(E - ½k × x₀²) × 2/m]
```

### P4.6 — Régimes avec frottements visqueux
```
Force : f = −h × v

h faible → courbe (Cc) : pseudo-périodique (oscillations amorties lentement)
h élevé  → courbe (Cd) : apériodique (pas d'oscillations)

h₁ < h₂ → (Cd) correspond à h₂
Justification : "Plus h est grand, plus l'amortissement est fort.
                 La courbe (Cd) revient à l'équilibre sans osciller → h₂."
```

Ka = 10^(−2pH) / (C − 10^(−pH))

Estérification :
  τf = xf / xmax
  K = xf² / [(n₁−xf)(n₂−xf)]
  Mélange équimolaire : τ'f = √K / (1+√K)

Piles :
  E° = E°_cathode − E°_anode
  K = 10^(E°/0,03)   [à 25°C]
  yéq = C(K−1)/(K+1)
```

## PHYSIQUE
```
Condensateur :
  i = C × du_C/dt   ;   τ = RC   ;   C = τ/R
  Charge   : u_C(t) = E(1 − e^(−t/τ))
  Décharge : u_C(t) = E × e^(−t/τ)

Bobine :
  u_L = L(di/dt) + ri   ;   τ = L/(R+r)

Oscillateur RLC :
  T₀ = 2π√(LC)   ;   L = T²/(4π²C)
  E = ½Li² + ½Cu_C²

Filtres :
  G = 20 log T
  Bande passante : T ≥ T₀/√2   ou   G ≥ G₀ − 3 dB
  Passe-bande : N₀ = 1/(2π√LC)  ;  ΔN = (R+r)/(2πL)
  Passe-bas   : N_C = 1/(2πRC)
  Passe-haut  : N_C = 1/(2πR₃C₂)

Ondes mécaniques :
  v = λN = λ/T   ;   N = 1/T
  y_M = a sin(2πNt − 2πx_M/λ + φ)   [pour t ≥ x_M/v]

Oscillateur mécanique :
  ω₀ = √(k/m) = 2π/T₀
  k = 4π²m/T₀²   (π² ≈ 10)
